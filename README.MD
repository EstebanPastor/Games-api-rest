# Games API REST 🎮

API REST desarrollada con **Node.js**, **Express** y **PostgreSQL** para la gestión de un catálogo de videojuegos.

## 🚀 Características
- **CRUD Completo**: Create, Read, Update y Delete.
- **Validaciones**: Protección contra tipos de datos inválidos (NaN), campos faltantes y control de errores de base de datos.
- **Seguridad**: Manejo de variables de entorno con `dotenv` y configuración de `CORS`.
- **Arquitectura**: Separación de responsabilidades en carpetas (`config`, `routes`, `controllers`).

## 🛠️ Tecnologías
- **Runtime**: Node.js
- **Framework**: Express
- **Base de Datos**: PostgreSQL (Librería `pg`)
- **Entorno**: Dotenv, Cors

## 📋 Requisitos Previos
- Node.js (v14+)
- Instancia de PostgreSQL activa (local o en la nube como AWS RDS).
